# Brand assets

This project expects the approved brand files at these exact public paths:

```text
public/brand/creotech-logo.png
public/brand/ritsumeikan-logo.jpg
```

Usage rules:

- Creotech is the primary operating brand for this support intake service.
- Ritsumeikan is shown only as a modest related/supporting institutional mark.
- The site should not appear to be the official Ritsumeikan University website.
- Do not replace these files without project-owner approval.
- Do not recreate placeholder logos.
- Do not stretch or distort logos.
- Render both logo images with `object-contain` and descriptive alt text.
